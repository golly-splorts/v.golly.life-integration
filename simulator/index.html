To start the simulator, click the Run button. Click again to pause it. The Step button will advance the simulator by a single step when it is paused. Refresh the window to reset the simulator.

You can also control the appearance of the simulator. Use the Colors button to cycle through different color schemes for the two teams. Use the Grid button to turn the grid on and off.

This simulator is a two-color version of Conway's Game of Life.
